# SBS Data Factory — Drive-to-Store DSP

Plateforme web DSP Drive-to-Store développée dans le cadre du stage PFA chez SBS Data Factory.

## Objectif

Développer une plateforme web permettant aux annonceurs de créer, gérer et suivre des campagnes publicitaires drive-to-store.

## Modules principaux

- Authentification et gestion des rôles
- Dashboard d'accueil
- Création de campagne
- Sélection des magasins
- Géofencing
- Créations / DCO
- Reporting
- Gestion du compte

## Stack technique prévue

- Frontend : React + Tailwind CSS
- Backend : FastAPI
- Base de données : PostgreSQL
- Visualisation : Recharts
- Cartographie : Google Maps API ou Mapbox
